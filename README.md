# BizPro - Business Location Intelligence

BizPro is a full-stack project that helps identify profitable business locations by analyzing factors like rent, competition, and customer flow.

## Features
- Business location analysis
- Data-driven insights
- Interactive frontend interface

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Data: CSV-based analysis

## How to Run

1. Clone the repository
2. Install dependencies:
   npm install

3. Start backend:
   node index.js

4. Open frontend in browser
